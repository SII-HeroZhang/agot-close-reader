# *A Game of Thrones* 逐章精读

## 阅读规则

- 每章一个独立 Markdown，按原 PDF 段落顺序逐段讲解。
- 所有人物姓名保留原始英文，不使用中文音译。
- 难词提供音标、词性、语境释义与用法提示。
- 背景与伏笔只使用读到当前段落时可知的信息，不泄露后文章节。
- 每个段落都有稳定编号和 PDF 页码，可用于后续提问。
